# How to analyze

## 1. Run the sample workloads

Optionally, run the F# sample projects:

dotnet run --project src/StackProcessing.RunSamples/RunSamples.fsproj -- --repeat 3 -j 6

Run the Studio JSON sample graphs:

dotnet run --project src/StackProcessing.RunSamples/RunSamples.fsproj -- --json --repeat 3 -j 6

## 2. Run Probe

Probe analyzes the sample measurements, freezes reliable coefficients, emits
designed calibration probes, runs those probes, repeats the analysis, and writes
the final sample estimates.

dotnet run --project src/StackProcessing.Probe/StackProcessing.Probe.fsproj -- calibrate --iterations 5 --run-probes --probe-repeats 3
